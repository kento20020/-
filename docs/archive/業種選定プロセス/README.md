# アーカイブ：業種選定プロセス

このフォルダの文書は、**本ゲーム開発とは直接関係しない過去の検討資料**です。
「限られたリソースでどの事業領域に取り組むか」を、課題定義→分解→アナロジー→発散→収束→
最小検証、という思考フレームで詰めた演習の記録です。

リポジトリ初期にアップロードされた経緯を保存するために残置しています。
ゲーム開発の参照先ではありません（設計は `../../設計仕様書.md` を参照）。
